    • 2012/02/02

      • Autostart option.
      • Links target rewriting option.
      • SITEINFO updater and manager.
      • IFRAME mode toggle.
      • Fast history navigation.
      • Support of custom CSS for the page separator and statusbar.
      • Support of custom SITEINFO patterns in JSON format.
      • Option to replace location URL on pagerizing.
      • Scroll height option.
      • Debug output support.
      • Site filters by regexps, prefixes and domains.
      • SITEINFO manager: instant search, editing and disabling of sites, internal auto-pagerizing.
      • Status bar: loading animation, toggle APW, call SI-manager or APW's preferences page.

      2012/19/06

      • Following custom SITEINFO fields were added: nextMask, pageElementSelector, nextLinkSelector, forceIframe, disableSeparator, forceAddressChange, allowScripts.
      • The link property is now available in AutoPatchWork.request event. It contains an element (Node type) with reference to the next page. Also there is norequest option (Boolean type) which defines whether or not to download the page referenced from previous element (if you set it to false blank page template will be returned).
      • Added the option to process next page before it is appended in AutoPatchWork.load event (including bringing it to the form specified in SITEINFO-pattern of the site). DOM-document of new pages is always available as htmlDoc (HTMLDocument type) property of this event.
